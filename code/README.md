# Code directory — active RABS workflow

Run from the project root.

## 1. Full-data RABS experiment

```bash
python3 code/run_rabs_adaptive_bandwidth.py
```

Reads:

```text
data/source/safety_probability_calibration_raw.csv
```

Writes:

```text
outputs/rabs/rabs_raw.csv
outputs/rabs/rabs_summary.csv
```

The current standard run uses all 8 available data windows.

## 2. Threshold tuning

```bash
python3 code/tune_rabs_fast.py
```

Writes:

```text
outputs/rabs_tuning/rabs_threshold_grid_fast.csv
outputs/rabs_tuning/rabs_tuned_summary.csv
```

Current tuned thresholds used in the main script:

```text
tau1 = 0.38
tau2 = 0.54
```

## 3. Figures from result CSVs

```bash
python3 code/make_rabs_figures.py
```

Writes data-driven SVG plots under:

```text
outputs/figures/
```

The AI overview image is not generated by this script; it is stored at:

```text
outputs/figures/fig_rabs_overview_ai.png
```
